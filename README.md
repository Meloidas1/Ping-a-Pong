# 🏓 Ping Pong Game

A modern and lightweight implementation of the classic **Ping Pong (Table Tennis)** arcade game built for smooth gameplay, real-time interaction, and clean game mechanics. This project demonstrates fundamental game development concepts such as collision detection, input handling, and frame-based animation.

---

## 📌 Overview

This project recreates the classic Ping Pong experience in a digital format. Two players control paddles on opposite sides of the screen, competing to keep the ball in play and score points against each other.

The focus of this project is on:
- Smooth and responsive gameplay
- Accurate collision physics
- Simple and intuitive controls
- Clean and modular code structure

---

## 🎮 Features

- Real-time paddle movement
- Smooth ball physics and bouncing mechanics
- Collision detection system
- Score tracking system
- Simple restart functionality
- Lightweight and fast performance
- Clean and minimal UI design

---

## 🎯 How to Play

The goal is simple: prevent the ball from passing your paddle while trying to score against your opponent.

- Each missed ball gives the opponent a point
- The game continues until a win condition is reached
- Timing and reflexes are key to winning

---

## 🎮 Controls

### Player 1
- Move Up: `W`
- Move Down: `S`

### Player 2
- Move Up: `↑ Arrow`
- Move Down: `↓ Arrow`

---

## 🧠 Game Mechanics

- The ball starts at the center and moves in a random direction
- Each paddle collision changes ball direction
- The ball speed may increase over time for added difficulty
- Score updates automatically when a player misses the ball
